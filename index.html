DATA ANALYTICS PROJECT

Análise Inteligente sobre Nascidos Vivos no Brasil

Projeto de análise de dados desenvolvido para explorar padrões, tendências e indicadores relacionados aos nascimentos no Brasil, utilizando Power BI, modelagem analítica e visualização executiva.

GitHub Ver Dashboard

INTRODUÇÃO

Objetivo do Projeto

Este projeto foi desenvolvido com o objetivo de transformar dados públicos de nascimentos no Brasil em informações estratégicas, permitindo identificar padrões demográficos, tendências temporais e indicadores relevantes para análise social.

A solução foi construída utilizando Power BI, DAX e técnicas de visualização executiva para criar um dashboard moderno, interativo e orientado à tomada de decisão.

O projeto também demonstra competências em análise exploratória, modelagem de dados, storytelling analítico e construção de indicadores inteligentes.

ANÁLISES REALIZADAS

Principais Insights

📈 Tendência Temporal

Identificação da evolução dos nascimentos ao longo do tempo, destacando períodos de crescimento e queda.

🧠 Inteligência Analítica

Uso de medidas DAX avançadas para cálculo de médias, crescimento percentual e comportamento histórico.

🌎 Distribuição Regional

Comparação entre estados e regiões brasileiras, identificando diferenças relevantes nos indicadores.

📊 KPIs Estratégicos

Desenvolvimento de métricas executivas para análise rápida e suporte à tomada de decisão.

⚡ Dashboard Interativo

Navegação intuitiva com foco em experiência do usuário e visual moderno estilo Big Tech.

🔍 Storytelling de Dados

Construção de uma narrativa visual para facilitar interpretação e comunicação dos resultados.
